Flex Item Properties

There are four Flex Item properties we'll learning about in our class. These properties are: flex-grow, flex-basis, order, and the flex shorthand property.

These properties can only be used on child elements INSIDE Flex Containers. Also, these child elements must be direct descendants of the parent Flex Container; in other words, they won't work on grandchildren or their descendants.

Remember, to turn on flex behavior for a container, you must set display: flex; on the element with CSS.
